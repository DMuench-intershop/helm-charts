# Helm Charts of Intershop`s Service Offering

Ready to launch on Kubernetes using Kubernetes Helm.

## TL;DR
```bash
$ helm repo add intershop https://intershop.github.io/helm-charts
$ helm repo update
$ helm install my-release intershop/<chart>
```
